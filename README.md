# AWS ECS FastAPI Logistics Cloud Center

This project goal is to build a scalable, secure, and reproducible cloud architecture that follows modern DevOps practices.

## Project start on 7/1/2026 

Throughout this 90-day Cloud Engineering transition program, this project achieved the following key milestones:
	1.	Cloud Networking: Designed a VPC with public and private subnets, ensuring security isolation between the application and database layers.
	2.	IaC Automation (Terraform): Defined the entire AWS infrastructure as code, ensuring environment "Reproducibility."
	3.	Container Deployment: Resolved container image compilation issues across different CPU architectures (ARM64 vs. AMD64) and established a foundation for automated CI/CD deployment.
	4.	Security Best Practices:
•	Zero-Secrets Policy: Implemented os.getenv for environment variables and automated RDS password generation, ensuring no sensitive credentials are hardcoded.
•	Security Groups: Strictly enforced the principle of least privilege for inbound/outbound traffic.
	5.	Troubleshooting: Mastered cloud-native debugging using Python log analysis and AWS CLI, resolving database connectivity timeouts and deployment configuration errors.
